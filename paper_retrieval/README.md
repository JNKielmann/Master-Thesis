**Folder structure**
- evaluation_notebooks:   
contains jupyter notebooks used for training and evaluating all models
- expert_voting:  
contains the expert voting code (document score modifier, aggregation functions, author score modifier)
- preprocessing:  
contains the code of the preprocessing pipeline
- retrieval_algorithms:  
contains the implementation of the used paper retrieval algorithms 
- server:  
contains code for running a server that provides access to the best model that was found
